📐 Modelo Matemático Aplicado

Intensidad Lumínica: I(r) = P / (r² + h²)
Derivada: I'(r) = -2Pr / (r² + h²)²
Condición de Optimización: I'(0) = 0 → Máximo en r = 0
Separación Calculada: d = 2√(P/I_min - h²)

Análisis: A mayor altura (h), menor intensidad máxima pero mayor cobertura. La eficiencia del sistema mejora porque se requieren menos postes para cubrir la misma área, manteniendo el umbral mínimo de iluminación.
